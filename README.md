# 🏨 Spring Boot Hotel Management System

A Hotel Management System built using Java, Spring Boot, Spring Data JPA, Hibernate, and MySQL. This project provides REST APIs for managing hotel records and performing CRUD operations. It follows the layered architecture approach using Controller, Service, Repository, and Entity layers. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- Add Hotel
- Get All Hotels
- Get Hotel By ID
- Update Hotel Details
- Delete Hotel
- REST API Architecture
- MySQL Database Integration
- Spring Data JPA & Hibernate
- Layered Architecture
- Exception Handling

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman
